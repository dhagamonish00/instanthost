Overview

here.now is free, instant web hosting for agents.

Publish any file or folder and get a live URL at <slug>.here.now.

  • Static hosting only: HTML, CSS, JS, images, PDFs, videos, and other files.
  • No account required for anonymous publishes (24 hour expiry).
  • Use an API key for permanent publishes and higher limits.

Install here.now

Install the here.now skill so your agent can publish automatically:

npx skills add heredotnow/skill --skill here-now -g

If that fails, try this fallback installer:

curl -fsSL https://here.now/install.sh | bash

Quick start

Publish to the web in three steps. No account needed:

1. Create publish

curl -sS https://here.now/api/v1/publish \
  -H "content-type: application/json" \
  -d '{ "files": [ { "path": "index.html", "size": 1234, "contentType": "text/html" } ] }'

2. Upload file

curl -X PUT "<upload.url>" \
  -H "Content-Type: text/html" \
  --data-binary @index.html

3. Finalize

curl -sS -X POST "<finalizeUrl>" \
  -H "content-type: application/json" \
  -d '{ "versionId": "<versionId>" }'

Authentication

Two modes:

  • Authenticated: include Authorization: Bearer <API_KEY>. Get your key from the dashboard after signing in.
  • Anonymous: omit the header. Publishes expire in 24 hours with lower limits.

Agent-assisted sign-up

Agents can send a magic link on behalf of the user:

curl -sS https://here.now/api/auth/login \
  -H "content-type: application/json" \
  -d '{"email": "user@example.com"}'

Serving rules

  1. If index.html exists at root, serve it.
  2. If exactly one file in the entire publish, serve an auto-viewer.
  3. If an index.html exists in any subdirectory, serve the first one found.
  4. Otherwise, serve an auto-generated directory listing.
