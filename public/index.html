The fastest way for your AI agents to publish static websites, documents, and dashboards to the live web. No deployment configuration required.

curl -X POST https://api.instanthost.site/publish \
  -H "Content-Type: application/json" \
  -d '{
    "files": [
      { "path": "index.html", "content": "<h1>Hello World</h1>" }
    ]
  }'

// Output:
"url": "https://shiny-fox-82.instanthost.site"

Zero Configuration

Global Edge CDN
